## Eventos

- on-change

```javascript
// jQuery
$('#element').on('change', function(){
  console.log("changed!! hehe");
});

// Javascript puro
document.getElementById('element').addEventListener('change', function(){
  console.log("changed!! hehe");
}, false);
```
